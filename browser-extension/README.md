# Phishing Detector

A real-time phishing detection system using machine learning and a browser extension.

## Setup

1. **Backend**:
   - Navigate to the `backend` folder.
   - Install dependencies: `pip install -r requirements.txt`
   - Run the Flask server: `python app.py`

2. **Browser Extension**:
   - Load the `browser-extension` folder in Chrome via `chrome://extensions/`.

## Deployment

- Deploy the Flask app on Heroku or any other cloud service.
- Update the `background.js` file with the deployed backend URL.
